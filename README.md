# Pixels News iOS

## Overview

In this app, you can see latest news, list news by topics and search news.

## Features

- Swift UIKit
- Storyboard
- UserDefaults
- WebView
- DarkMode

## Architech

- Protocol oriented MVVM.

## API

- News [API](https://newsapi.org/).
